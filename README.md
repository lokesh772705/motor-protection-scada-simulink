# ⚡ Industrial Motor Protection System with SCADA (MATLAB/Simulink)

## 🚀 Overview
This project is a simulation of a real industrial motor protection system built using MATLAB/Simulink.  
The goal was to understand how protection relays work in industries and to visualize everything through a SCADA-like dashboard.

---

## ⚙️ What this project does
- Simulates a **3-phase motor system (400V, 50Hz)**
- Continuously monitors **current and voltage**
- Detects faults in real time
- Automatically **trips the circuit breaker** to protect the motor
- Displays everything on an interactive **SCADA dashboard**

---

## 🔥 Key Features
- ⚡ Overcurrent protection (IDMT logic)
- 🔻 Under-voltage & 🔺 Over-voltage protection
- 🌍 Earth fault detection
- ⚖️ Phase imbalance detection
- 🔌 Automatic breaker tripping
- 📊 Live dashboard with gauges, indicators, and waveforms

---

## 🧠 How it works
The system uses a **MATLAB Function block** to act like a digital protection relay:
- Takes current & voltage as input
- Checks for fault conditions
- Generates a **trip signal**
- Identifies the type of fault

---

## 📊 SCADA Dashboard
The dashboard gives a real-time view of the system:
- Current & voltage readings
- Fault indication (lamps)
- Relay status (ON/OFF)
- Live waveform visualization

---

## 🧪 Results
- Fast response time (~0.5 seconds)
- Reliable detection of multiple faults
- Smooth integration of protection + visualization
